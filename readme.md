<p>1. Downloading the Code</p>

-  Open your preferred source code editor and using the terminal, type "git clone jiuuzxc/Flu-Simulation-Model.git" to create remote access.


```
git clone jiuuzxc/Flu-Simulation-Model.git
```


<p>2. Required Libraries</p>

- Install packages first in terminal, using the command: "pip install -r requirements.txt".
- Heres the list of the Libraries to download: numpy,pandas,scikit-learn,flask,flask-cors,xgboost


```
pip install -r requirements.txt
```

<p>4. Running the Server</p>

- Change directory to the repository folder and Run the flaskapi.py for the server.

<p>5. Running the Code</p>

- Use this command on your terminal to run the Flu Simulation Code.

```
"python -m http.server
```
```
Enter "http://localhost:8000/" in your browser.
```
