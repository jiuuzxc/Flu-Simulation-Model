Libraries to download (pip install)

numpy
pandas
scikit-learn
flask
flask-cors
xgboost

STEPS TO RUN THE PROGRAM:

1. Install packages first in terminal, using the command: "pip install -r requirements.txt"

2. Run the flaskapi.py for the server.

3. Enter "python -m http.server" to your terminal for the local host.

4. Enter "http://localhost:8000/" in your browser.
